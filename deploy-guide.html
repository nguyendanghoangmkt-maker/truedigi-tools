🚀 Hướng Dẫn Deploy Online

Đưa TrueDigi Tools lên GitHub Pages → tools.truedigi.com (không cần lập trình)

1

Tạo Repo GitHub & Upload Files

~10 phút · Không cần Git · Chỉ cần browser

1
Tạo repo mới trên GitHub
Vào github.com/new (đăng nhập bằng tài khoản nguyendanghoangmkt-maker).
Đặt tên repo: truedigi-tools → Chọn Public → nhấn Create repository.
2
Upload toàn bộ file HTML
Trong trang repo vừa tạo → nhấn "uploading an existing file" hoặc "Add file → Upload files".
Kéo thả toàn bộ files trong thư mục G:\Other computers\My PC\D\truedigi\ vào đây.
⚠️ Cần upload những file này: agency-hub.html, brief-nhanh.html, proposal-builder.html, client-tracker.html, bao-cao-thang.html, ai-content-plan.html, bao-gia-dao-tao-ai.html, CNAME, index.html, 404.html

Và thư mục: in nhanh/ (kéo cả thư mục vào)
Nhấn Commit changes.
3
Bật GitHub Pages
Vào repo → Settings → thanh trái chọn Pages.
Phần Source: chọn Deploy from a branch → Branch: main → folder: / (root) → Save.
✅ Sau 2-5 phút: URL sẽ là https://nguyendanghoangmkt-maker.github.io/truedigi-tools
2

Gắn Domain tools.truedigi.com (Mắt Bão)

~5 phút · Cần đăng nhập Mắt Bão · Chờ 15-30 phút propagate

1
Thêm DNS record tại Mắt Bão
Đăng nhập matbao.net → Quản lý Tên Miền → truedigi.com → DNS Management.
Thêm record mới:
Type Host/Name Value/Target TTL
CNAME tools nguyendanghoangmkt-maker.github.io. 3600
Nhấn Save.
2
Cấu hình GitHub Pages custom domain
Quay lại GitHub → Settings → Pages → phần Custom domain.
Điền: tools.truedigi.com → Save.
Chờ DNS propagate (15–60 phút) → tick Enforce HTTPS.
✅ Kết quả: https://tools.truedigi.com → Agency Hub
3

Cài 4 Automation Scripts (Google Apps Script)

Miễn phí · Chạy tự động trên Google Cloud

📌 Cách mở Apps Script: Vào script.google.com → New Project → đặt tên "TrueDigi Automation" → paste code vào.
📧 Script 1: Gửi Báo Cáo Tháng
Nhận data từ bao-cao-thang.html → format email đẹp → gửi cho client
apps-script/script1-email-report.gs
Setup: Paste code → chạy menuSetup() → Deploy → Web App → Copy URL → dán vào bao-cao-thang.html
🔔 Script 2: Nhắc Deadline Mỗi Sáng 8h
Mỗi sáng tự kiểm tra deadline, gửi email nhắc cho team & account phụ trách
apps-script/script2-deadline-reminder.gs
Setup: Đổi NOTIFY_EMAIL → chạy menuSetupDeadlines() → Triggers → Add: checkDeadlines → Day timer → 8AM
📊 Script 3: KPI Auto-Pull (GA4 + Meta Ads)
Mỗi sáng 7h kéo data từ GA4 và Facebook Ads về Google Sheets tự động
apps-script/script3-kpi-pull.gs
Setup: Điền GA4 Property ID → Enable GA4 Data API trong Google Cloud → menuSetupKPI() → Trigger: pullKPIDaily → 7AM
🤖 Script 4: AI Content Plan (Gemini)
Đầu mỗi tháng tự generate content plan cho tất cả client → lưu vào Sheets
apps-script/script4-ai-content.gs
Setup: API key đã điền sẵn → menuSetupContent() → testGemini() để test → Trigger: autoGenerateMonthlyPlans → Month timer → day 1
✅ Tool AI Content Plan HTML File ai-content-plan.html đã có sẵn và hoạt động không cần Apps Script — mở file là dùng được ngay, Gemini API key đã tích hợp sẵn.
